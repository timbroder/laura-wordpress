Visio Theme

Readme file of Visio Theme

  1. Upload "Visio" folder to themes directory (/wp-content/themes/)

  2. Upload plugin on "plugins" folder to plugins directory (/wp-content/plugins/) and activated

  3. After Install the theme, go to "Visio Options" (Appearance -> Visio Options)

  4. And than, you can see Visio Options

  5. If you want add image on "sliding image", don't forget or "Costum Fields" when you create post
    put "thumbnail" for name and image path (URL) for value.

    For best resolution (image) use "201px * 157px"

For more information go to our forum
